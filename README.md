🗺️ GeoNotes — карта заметок с геолокацией

> Интерактивное приложение для создания заметок с привязкой к географическим координатам.  
> Отображайте свои заметки на карте, ищите по местоположению, делитесь маршрутами.  
> Реализовано на 7 языках программирования.

## 🧠 Концепция

**GeoNotes** — это не просто заметки. Это ваш личный дневник мест:

- ✅ **Добавление заметок** с текстом, координатами (широта/долгота) и категориями.
- ✅ **Отображение на карте** (интеграция с OpenStreetMap/Leaflet или Google Maps).
- ✅ **Определение текущего местоположения** (через GPS или IP).
- ✅ **Поиск по радиусу** — покажите все заметки в пределах N км от заданной точки.
- ✅ **Список заметок** с сортировкой по дате или расстоянию.
- ✅ **Экспорт/импорт** в JSON и GPX (опционально).
- ✅ **Тёмная тема** (в GUI-версиях).
- ✅ **Геокодирование** — преобразование адреса в координаты и обратно.

## 🚀 Функциональность

| Команда / Действие          | Описание |
|-----------------------------|----------|
| `add "текст" [lat] [lon]`   | Добавить заметку (если координаты не указаны, используется текущее местоположение). |
| `list`                      | Показать все заметки. |
| `map`                       | Открыть карту с заметками в браузере (или встроенном виджете). |
| `search <текст>`            | Найти заметки по тексту. |
| `near <lat> <lon> <radius>` | Найти заметки в радиусе (км) от точки. |
| `delete <id>`               | Удалить заметку. |
| `edit <id> "новый текст"`   | Изменить текст заметки. |
| `export <file.json>`        | Экспортировать все заметки в JSON. |
| `import <file.json>`        | Импортировать заметки из JSON. |
| `geocode "адрес"`           | Получить координаты по адресу (через Nominatim). |
| `reverse <lat> <lon>`       | Получить адрес по координатам. |

## 📦 Содержимое репозитория

| Файл                  | Язык       | Особенности |
|-----------------------|------------|-------------|
| `geonotes_python.py`  | Python     | + Tkinter GUI + встроенная карта через `folium` (открывается в браузере) |
| `geonotes_cpp.cpp`    | C++        | + Qt Widgets + WebEngineView для отображения карты (Leaflet) |
| `geonotes_java.java`  | Java       | + Swing + WebView (JavaFX или JxBrowser) – упрощённо, открываем браузер |
| `geonotes_cs.cs`      | C#         | + WPF + WebBrowser (или CefSharp) для карты |
| `geonotes_go.go`      | Go         | + консоль, генерация HTML-карты и открытие в браузере |
| `geonotes_rs.rs`      | Rust       | + консоль, генерация HTML-карты через `handlebars` |
| `geonotes_js.js`      | JavaScript | + Electron + Leaflet, полноценное настольное приложение |

## 🛠️ Как запустить

Каждая версия требует установки соответствующих библиотек/инструментов:

```bash
# Python
pip install folium geopy requests tkintermapview  # опционально
python geonotes_python.py

# C++ (Qt)
qmake && make
./geonotes_cpp

# Java (для WebView требуется JavaFX)
javac -cp . geonotes_java.java && java geonotes_java

# C# (WPF)
dotnet add package Microsoft.Web.WebView2
dotnet run

# Go
go mod tidy
go run geonotes_go.go

# Rust
cargo build --release && ./target/release/geonotes_rs

# Node.js (Electron)
npm install
npm start
